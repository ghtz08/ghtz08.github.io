# SCSS 简介

Sass 是一种 CSS 预处理器，这个预处理器包含两种语法：

- Sass：一种缩进语法
- SCSS：一种 CSS-like 语法

# SCSS 基础语法

- 变量用 `$` 表示  
- 导入 `@import`  
- 混合 `@mixin`  
- 继承 `@extend %`
- 操作符 `+, -, *, /, %`
- 引用父级选择器 `&`  
- 嵌套属性 `: {`
